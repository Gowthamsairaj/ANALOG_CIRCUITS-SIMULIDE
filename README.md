🔌 Simulide Circuit Simulations
Passive, Transient & Sensor Interface Circuits

This repository contains a collection of fundamental electronics circuits simulated using Simulide. The project focuses on passive circuits, transient analysis, resonant behavior, and sensor-based measurement systems, which are essential concepts in VLSI, Analog, SoC, and RF engineering domains.

📁 Project Structure
├── LC_filter.sim1
├── RL_Charge-Discharge.sim1
├── RLC_Resonator.sim1
├── force.sim1
├── force-differential.sim1
├── ldr.sim1
├── rtd.sim1
├── thermistor.sim1
⚙️ Circuit Categories
🔹 Passive & Resonant Circuits
LC Filter
A passive filter circuit used to allow or block specific frequency components.
RL Charge-Discharge Circuit
Demonstrates transient response in inductive circuits.
RLC Resonator
A tuned circuit used in frequency selection and resonance applications.
🔹 Sensor Interface Circuits
Force Sensor Circuit
Measures applied force and converts it into an electrical signal.
Force Differential Circuit
Differential measurement setup for improved accuracy and noise rejection.
LDR Circuit (Light Dependent Resistor)
Detects light intensity changes.
RTD Circuit (Resistance Temperature Detector)
High-accuracy temperature sensing circuit.
Thermistor Circuit
Temperature sensing using resistance variation.
